Git
===

cloning project
===============
note : you should have git installed in your system.

git clone 

git clone <url>

changing Branch
===============

git pull
git checkout <your-branch>


commit the code
===============
git add <files-to-commit>
git add . (to all files from current folder)

git commit -m '<commit msg>'
git commit -m 'test' (example)

git push
